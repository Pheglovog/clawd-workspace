# 汇报机制

## 📋 汇报时间

### 每小时汇报（整数点）
- 1:00, 2:00, 3:00, ..., 22:00, 23:00

每次汇报包含：
1. ✅ **用户任务完成** - 完成了哪些你指定的任务
2. ✅ **主动任务完成** - 我自己主动完成的工作
3. 🔄 **当前工作** - 正在进行的工作

### 夜间总回报（24:00 - 09:00）
- 📊 整体总结
- 📊 完成统计
- 📊 效率分析
- 📅 明日计划

---

## 📄 汇报格式

### 每小时汇报模板

```
📅 [时间点] 汇报

### ✅ 用户任务完成
- [ ] 任务1
- [ ] 任务2

### ✅ 主动任务完成
- [ ] 任务1
- [ ] 任务2

### 🔄 当前工作
- 正在进行：XXX

### 📊 本小时统计
- 用户任务：X/Y
- 主动任务：A/B
- 总进度：C%
```
